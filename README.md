# Sistem za praćenje košarkaške NBA lige

## O projektu
Ovaj projekat predstavlja web stranicu za praćenje NBA lige. Stranica omogućava pregled NBA timova, tabela konferencija, statistike igrača, video sadržaja, zanimljivosti i interaktivne mape NBA arena. Projekat je realizovan korištenjem HTML-a, CSS-a i JavaScript-a.
